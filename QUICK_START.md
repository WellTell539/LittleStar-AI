# 🚀 LITTLE STAR AI AI Dashboard - 快速开始指南

## 🎯 30秒快速启动

```bash
# 1. 克隆项目
git clone <项目地址>
cd ClaudeAI/claude-ai-dashboard

# 2. 安装依赖并启动
npm install && npm run dev

# 3. 打开浏览器
访问 http://localhost:3000
```

## 🎮 首次使用指南

### 第一步：认识你的AI伙伴
1. **观察AI状态** - 查看右上角的AI吉祥物，它会根据心情变化表情
2. **查看基本信息** - 在"概览"页面了解AI的基本人格和状态
3. **阅读当前心情** - 注意AI的情感状态和强度

### 第二步：开始对话
1. **点击"聊天"标签** - 进入AI对话界面
2. **发送第一条消息** - 试着说"你好"或"今天心情怎么样？"
3. **观察AI反应** - 注意AI的个性化回复和情感表达

### 第三步：设置目标
1. **访问"目标"页面** - 为AI设定学习或活动目标
2. **选择目标类型** - 学习、创作、反思等不同类型
3. **设定时间** - 可以精确到分钟的时间安排

### 第四步：查看记忆
1. **打开"记忆"管理** - 查看AI的记忆和经历
2. **添加重要记忆** - 手动添加重要的对话或事件
3. **查看记忆统计** - 了解AI的记忆分布和重要性

### 第五步：体验高级功能
1. **AI洞察** - 查看AI的学习进度和情感分析
2. **生活模拟** - 观看AI的日常活动模拟
3. **Web3功能** - 连接钱包体验区块链功能（可选）

## 🆘 常见问题

### Q: AI没有回应怎么办？
A: 检查控制台是否有错误，尝试刷新页面或重启开发服务器

### Q: 如何重置AI状态？
A: 在浏览器开发者工具中清除 localStorage，或在设置页面点击重置

### Q: Web3功能无法使用？
A: 确保已安装MetaMask等Web3钱包，并连接到正确的网络

### Q: 页面加载缓慢？
A: 检查网络连接，清除浏览器缓存，或使用生产版本（npm run build && npm run start）
